# View The App

This project is a React + Vite app, so the main file to open is not a standalone finished page.

## Important

Opening `index.html` directly in the browser is usually not enough for a Vite React app.
The app should be started through the Vite dev server.

## Project files you can open

- `src/App.jsx` - main app logic
- `src/components/Toolbar.jsx` - control buttons
- `src/components/GridBoard.jsx` - grid rendering
- `src/style.css` - styling

## To run the app locally

From this project folder, run:

```powershell
npm install
npm run dev
```

Then open the local URL shown in the terminal, usually:

```text
http://localhost:5173
```

## If `npm` is not installed

Install Node.js from the official website. `npm` is normally included with Node.js.

After that, reopen your terminal in this folder and run:

```powershell
npm install
npm run dev
```
